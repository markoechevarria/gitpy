# GitPy

A VCS (Version Source Control) written in python.

## Installation

1. Clone the repository
```
git clone https://github.com/markoechevarria/gitpy.git
```
2. Enter directory, set up the virtual environment and install dependencies
```
cd gitpy
python3 -m venv ven
source ./venv/bin/activate
```

3. Install dependencies
```
pip install .
```

## Usage

| Option    | Command   |
| ----      | --------- | 
| init      | `python3 src/gitpy/main.py init` |
| end       | `python3 src/gitpy/main.py end` |
| add       | `python3 src/gitpy/main.py add` |
